# Project Description

Newsbot for current events. Fetches articles from AP News website, identifies candidate articles, evaluates and extracts key information, uses LLM to generate summaries, emails summaries to users and cleans up temporary files. Creates artifacts for key steps for debugging and monitoring. Can run with Hermes agent, a local cron, or be triggered manually.

## Architecture
- **Data Flow**: web → json artifacts → LLM → md artifacts → email
